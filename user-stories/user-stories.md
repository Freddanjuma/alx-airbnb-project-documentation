## üí° User Stories

### 1. Guest Registration
**As a Guest**, I want to create an account so that I can log in and book properties on the platform.

### 2. Property Listing
**As a Host**, I want to list my property with details and images so that potential guests can find and book it.

### 3. Property Search & Booking
**As a Guest**, I want to search for available properties and make a booking so that I can reserve a place to stay.

### 4. Payment Processing
**As a Guest**, I want to make secure payments for my bookings so that my reservation is confirmed.

### 5. Admin Management
**As an Admin**, I want to manage user accounts and property listings so that the system remains trustworthy and compliant.

### 6. Host Earnings
**As a Host**, I want to view my booking history and earnings so that I can track my income.

### 7. Payment Confirmation (External)
**As a Payment System**, I want to confirm transactions back to the Airbnb system so that booking statuses update automatically.
